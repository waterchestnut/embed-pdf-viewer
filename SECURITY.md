# Reporting Security Issues

If you believe you have found a security vulnerability in EmbedPDF, we appreciate your help in disclosing it to us responsibly.

We take all reports seriously and will investigate them promptly. We are committed to working with you to understand and resolve any legitimate issues quickly.

Please report any potential vulnerabilities privately to **security@embedpdf.com**.

**Please do not** disclose security issues in public GitHub issues.
